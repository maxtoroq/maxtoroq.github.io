---
title: Installing
---

DbExtensions' binaries are hosted on [NuGet.org](https://www.nuget.org/packages/DbExtensions).

```powershell
Install-Package DbExtensions
```

Target frameworks
-----------------
See the table below for DbExtensions' framework support for each major version.

Version | Frameworks
------- | ----------
v6      | .NET 4.5
v5      | .NET 4.0
v4      | .NET 3.5, .NET 4.0
