... if the route doesn't have the {action} token, e.g.

routes.MapRoute(null, "",
   new { controller = @"home", action = @"index" },
   new[] { "Company.Website.Home" });

Must be changed to this:

routes.MapRoute(null, "{action}", 
   new { controller = @"home", action = @"index" }, 
   new { action = @"index" }, 
   new[] { "Company.Website.Home" });
